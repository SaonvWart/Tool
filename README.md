# Tool - 多功能生成工具箱

一个功能不强大的Python命令行工具集，包含多种实用工具，从编码解码到卡密生成，从加密解密到格式转换。

## 主要特性

- **多功能集成**: 集成20+种实用工具
- **模块化设计**: 每个工具独立运行，互不干扰
- **易于使用**: 简洁的菜单界面，支持中文/英文
- **可扩展**: 支持自定义格式和模板

## 功能列表

### 1. 卡密生成器
- 支持多种格式模板（魔理沙、UUID、Drip、雪碧、MinecraftUnBan等）
- 可生成带账号信息的Minecraft卡密
- 入侵卡网动画效果
- 高级配置选项（订单号、格式、分组等）

### 2. 编码/解码工具
- Hex编码/解码
- Base64编码/解码
- URL编码/解码

### 3. 加密工具
- RSA加密/解密/签名/验签
- AES-GCM加密/解密
- 国密SM2加密/解密
- CSR证书生成/查看

### 4. 哈希工具
- MD5、SHA1、SHA256、SHA512等哈希计算
- 文件和字符串哈希

### 5. 生成工具
- UUID生成器（可自定义格式）
- HWID生成器
- 随机账号生成（邮箱、密码、Minecraft账号）
- 漫画图片生成器（A4模板）

### 6. 格式转换
- JSON美化/压缩
- JavaScript美化/压缩
- HTML、CSS、YAML、XML格式化
- 图片格式转换

### 7. 其他工具
- 二维码/条形码生成
- 4399账号转Cookie
- Google搜索/翻译
- Python代码编译
